Android笔记-Intent

Intent的一些基本用法

Intent作用之一是用于Activity之间的来回切换（），是一个Activity启动另外一个必须要使用的组件。分为显式和隐式。
1.显式Intent
使用例子如下：

Button btn1 = (Button)findViewById(R.id.btn1);
btn1.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
        Intent intent = new Intent(MainActivity.this, Activity2.class);
        startActivity(intent);
    }
});

2.隐式Intent
使用例子如下：

    findViewById(R.id.btn2).setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View v) {
            startActivity(new Intent("io.github.wwy941122.intenttest.intent.ACTION"));
        }
    });
}

此时只需要知道Activity在Manifests中的名称就可以启动。

<intent-filter>
    <action android:name="android.intent.action.MAIN" />   //此处的名称

    <category android:name="android.intent.category.LAUNCHER" />
</intent-filter>

Intent在两个Activity过程中还可以传递数据，用到putextra()、getextra()这两个个方法，可传递String和Bundle等

另外上文分别展示了两种Button点击事件响应方式。
