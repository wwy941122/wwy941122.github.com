markdown语法

一些常用markdown语法，以便查阅

注意

标题冒号后有空格！！！！！

Markdown 文档

标题

1.在标题下发加===或—

第一种标题
====

第二种标题
---

2.标题前加#号

# 标题1
## 标题2
### 标题3
#### 标题4
##### 标题5

引用

在文字前加> ，注意有空格。

> 引用

可以嵌套引用

> 引用
>> 引用2

列表

分为无序列表和有序列表

无序列表用+、*、-
- 1111
- 2222
- 3333
列表也可以嵌套

+ 1
- 111
- 222
+ 2
- 333
- 444

代码

发现两种方法, 风格有所不同

一种是`符号，注意这个不是引号而是~键的那个。
一种是在代码前加4个空格，markdown会自动转换

+ this is the first method

+ this is the second one

还有一种这种
可指定编程语言，『』代表左右大括号

『% codeblock [title] [lang:language] [url] [link text] %』
code snippet
『% endcodeblock %』

斜体和粗体

斜体
将需要设置为斜体的文字两端使用1个“*”或者“_”夹起来。

粗体
将需要设置为斜体的文字两端使用2个“”或者“_”夹起来。
*good

  1. 文字超链
    Github
  2. 自动链接
    尖括号<>
    http://www.baidu.com
