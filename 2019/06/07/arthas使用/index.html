arthas使用

功能包括：反编译，热更新代码，追踪方法调用路径和执行时间，监测方法的入参、返回值
https://alibaba.github.io/arthas/commands.html

命令

thread

查看线程信息
thread -n 3 查看Top3

trace

方法内部调用路径，并输出方法路径上的每个节点上耗时
trace com.xiaomi.youpin.kylin.rule.ConfigurationConflictRule checkConflict

watch

方法执行数据观测，能观察到的范围为：返回值、抛出异常、入参
watch com.xiaomi.youpin.kylin.rule.ConfigurationConflictRule buildResponse “{params[0].size(), params[2].size()}” -x 2 -b

monitor

方法执行监控
对匹配 class-pattern／method-pattern的类、方法的调用进行监控。
monitor 命令是一个非实时返回命令

反编译

反编译出class文件

jad --source-only com.example.demo.arthas.user.UserController > /tmp/UserController.java

查找加载UserController的ClassLoader

sc -d *UserController

找到classLoaderHash

内存编绎代码生成class文件

mc -c 49c2faae /home/work/wangweiyi/ConfigurationConflictRule.java -d /home/work/wangweiyi

热更新

redefine /tmp/com/example/demo/arthas/user/UserController.class
